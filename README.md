## How to Use

1. **Open** `index.html` in any web browser
2. **Enter** a city name (optional) - e.g., "Mumbai", "Delhi"
3. **Enter** your craving or mood - e.g., "spicy", "sweet", "tangy", "comfort"
4. **Click** "Get Recommendation" or press Enter
5. **Discover** authentic street food with location and description
